User Guide 
----------

Welcome to the ALC AiiDAlab app's user guide. This documentation is designed to help you get started with using the app effectively. 
Whether you're new to AiiDAlab or looking to enhance your workflow, you'll find valuable information here. 
This guide is intended for developers or "power users" who wish to tailor the configuration of AiiDAlab, 
for a basic startup guide see the 
`AiiDAlab documentation <https://aiidalab.readthedocs.io/en/latest/usage/access/index.html>`_ and for specific AiiDAlab 
plugins please refer to their individual documentation for user guides. Additionally, whilst some of the concepts of AiiDA 
are described throughout the documentation it is assumed that the user has a basis working understanding of what AiiDA and 
what it can provide. 
The first few sections detail the process of deploying/running AiiDAlab under different scenarios, if you simply want to run 
AiiDA/AiiDAlab without any extra configuration see the 
`Access AiiDAlab guide <https://aiidalab.readthedocs.io/en/latest/usage/access/index.html>`_\. The final section 
gives some guides to how to setup core AiiDA features from within the AiiDAlab application.
